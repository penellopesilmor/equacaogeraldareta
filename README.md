# Equação geral da reta
Programa em Python com objetivo de solucionar equações da reta.
# Objetivos:
Servir ao mesmo tempo de método de revisão matemática, ferramenta tecnológica e estudo das funções do pyhton.
# Funcionalidades:
Serão adicionadas funcionalidades para cada casp específico do estudo da reta e por fim um programa no qual o usuário pode navegar entre essas opçoes.
